##  项目简介

**人生 K 线** 是一个创新的个人日记应用，将人生状态类比为股票 K 线图，通过记录每日的心情、事件和涨跌幅，生成专属的人生 K 线走势。内置 3 个 AI Agent，在你人生的每个时刻给予温暖的鼓励和正确的引导。

##  核心特性

###  人生 K 线全景图
- **标准股票 K 线图**：支持日 K、周 K、月 K、年 K 切换
- **红涨绿跌配色**：经典股票配色，一目了然
- **历史数据补充**：可手动录入历史年份 K 线，完善人生轨迹
- **动态价格监测**：自动计算开盘价、收盘价、最高价、最低价

###  3 个专属 AI Agent

#### 1️ 情绪复盘 Agent（每日执行）
- 读取今日日记和涨跌幅
- 分析情绪强度，修正 K 线影线
- 输出当日情绪复盘报告

#### 2️ 趋势分析 Agent（周/月/年执行）
- 分析全部历史 K 线数据
- 判断人生阶段：筑底期/横盘震荡/主升浪/高位回调
- 预测下一阶段走势

#### 3️ 人生操盘 Agent（决策辅助）
- **大幅下跌时**：给予关心安慰，"今天一定很不容易吧，但请记住，这只是人生长河中的一个小波澜"
- **小幅下跌时**：温和鼓励，"今天的小幅回调是在积蓄力量，就像弹簧压缩后会弹得更高"
- **大幅上涨时**：热情鼓励 + 建议，"太棒了！今天的你闪闪发光，继续保持这份状态！"
- **小幅上涨时**：肯定进步，"稳步上涨是很好的状态！积小胜为大胜，持续的进步最可贵"
- **横盘震荡时**：鼓励沉淀，"利用这段时间沉淀自己，学习新技能，等待属于你的突破时机"

###  视觉设计
- **黑白极简风格**：高级深色科技风
- **鼠标交互效果**：移动鼠标时颗粒变色（黑→白→黑）
- **玻璃态 UI**：半透明背景 + 模糊效果
- **响应式设计**：完美适配手机、电脑

###  数据存储
- **本地存储**：所有数据存储在浏览器 localStorage
- **无需后端**：打开网页即可使用
- **导入导出**：支持 JSON 格式备份和恢复
- **刷新不丢失**：数据持久化保存

##  快速开始

### 方式一：直接打开
1. 下载 `life_kline_v2.html` 文件
2. 双击用浏览器打开
3. 开始记录你的人生 K 线！

### 方式二：本地服务器
```bash

python -m http.server 8000

# 使用 Node.js
npx serve .
```

然后访问 `http://localhost:8000`

##  设计理念

> **人生如 K 线，有涨有跌才是常态**

我们相信：
- 每一天都值得记录
- 每一次起伏都有意义
- 低谷时需要关心安慰
- 高峰时需要鼓励肯定
- 平淡时需要发现价值

##  功能亮点

### 1. 日期重复检测
- 年 K 线：同一年份不能重复添加
- 日 K 线：同一日期不能重复添加
- 编辑时自动排除当前记录

### 2. 历史数据编辑
- 点击"编辑"自动填充表单
- 修改后重新生成 AI 分析
- 支持取消编辑

### 3. 动态价格监测
- 初始值：100（人生发售价）
- 开盘价：最新记录的收盘价
- 包含手动录入的年 K 线

### 4. 数据隔离
- 手动录入的年 K 线只在年 K 图显示
- 日 K、月 K、周 K 不显示年 K 线
- 保持图表清晰度

##  特色功能

### 温暖 AI 建议
完全去除金融术语，改用：
-  关心安慰（下跌时）
-  鼓励肯定（上涨时）
-  正向引导（震荡时）
-  人生化比喻（弹簧、积蓄力量等）

### 随机化表达
同一种情况有多种不同的温暖话语，避免重复枯燥

# 人生 K 线 - Electron 桌面版使用指南

恭喜！您的人生 K 线应用现在有了 Electron 桌面版本，数据将永久保存在本地文件中。

## 完成的工作

### 已创建的文件

1. **main.js** - Electron 主进程文件
2. **preload.js** - 预加载脚本，提供安全的 API 桥接
3. **package.json** - 项目配置和依赖管理
4. **.gitignore** - Git 忽略文件配置
5. **start.bat** - Windows 快速启动脚本
6. **build.bat** - Windows 快速打包脚本
7. **INSTALL.md** - 详细安装指南
8. **life_kline_v2.html** - 已升级为智能存储版本

### 核心功能

1. **智能存储系统**
   - Electron 环境：数据保存在本地 JSON 文件
   - 浏览器环境：数据保存在 localStorage
   - 自动检测环境，无缝切换

2. **数据持久化**
   - 数据文件位置：`%APPDATA%\人生 K 线\data\life_kline_data.json`
   - 不受浏览器缓存清除影响
   - 每次保存自动写入磁盘

3. **系统级对话框**
   - 导出：使用系统文件保存对话框
   - 导入：使用系统文件选择对话框
   - 更原生、更好用的体验

## 快速开始

### 方式一：双击启动（最简单）

1. 双击 `start.bat` 文件
2. 首次运行会自动安装依赖
3. 应用自动打开

### 方式二：命令行启动

```bash
# 安装依赖（仅首次）
npm install

# 启动应用
npm start
```

## 打包成可执行文件

### Windows:
```bash
npm run build:win
```
生成的文件：`dist/人生 K 线 Setup x.x.x.exe`

### macOS:
```bash
npm run build:mac
```
生成的文件：`dist/人生 K 线-x.x.x.dmg`

### Linux:
```bash
npm run build:linux
```
生成的文件：`dist/人生 K 线-x.x.x.AppImage`

## 数据迁移

### 从浏览器版迁移到桌面版

1. 在浏览器中打开旧版本
2. 点击"历史数据" → "导出 JSON"
3. 保存 JSON 文件
4. 打开 Electron 桌面版
5. 点击"历史数据" → "导入数据"
6. 选择之前导出的 JSON 文件
7. 数据迁移完成！

## 数据备份建议

虽然数据已经保存在本地文件中，仍建议定期备份：

1. **每周导出**：使用应用内的导出功能
2. **云同步**：将导出的 JSON 文件保存到云盘
3. **外部存储**：定期备份到 U 盘或移动硬盘

## 优势对比

### vs 浏览器版

| 特性 | 浏览器版 | Electron 桌面版 |
|------|----------|----------------|
| 数据存储 | localStorage | 本地 JSON 文件 |
| 数据安全性 | 清除缓存会丢失 | 永久保存 |
| 启动方式 | 需要打开浏览器 | 双击即可 |
| 使用体验 | 网页体验 | 原生应用体验 |
| 跨设备 | 数据不互通 | 可拷贝数据文件 |

###  vs 其他桌面应用

| 特性 | 传统桌面应用 | 人生 K 线 Electron 版 |
|------|--------------|---------------------|
| 安装大小 | 几十 MB | 约 100MB |
| 启动速度 | 快 | 快 |
| 数据格式 | 专有格式 | 开放 JSON 格式 |
| 可移植性 | 差 | 好（可复制数据文件） |
| 可维护性 | 困难 | 简单（HTML 文件） |

### 文件保存路径

- **Windows**: `C:\Users\用户名\AppData\Roaming\人生 K 线\data\`
- **macOS**: `~/Library/Application Support/人生 K 线/data/`
- **Linux**: `~/.config/人生 K 线/data/`

##  常见问题

### Q: 数据文件在哪里？
A: 在系统的应用数据目录中，具体路径见上方"文件保存路径"。

### Q: 可以手动编辑数据文件吗？
A: 可以，数据是 JSON 格式，但建议先备份再编辑。

### Q: 如何完全卸载？
A: 删除应用和数据目录即可。Windows 可在控制面板卸载。

### Q: 可以在多个电脑使用吗？
A: 可以，复制数据文件到其他电脑即可同步数据。

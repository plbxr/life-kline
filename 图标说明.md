# 图标文件说明

## 需要的图标文件

为了打包成桌面应用，您需要准备以下图标文件：

### Windows:
- `icon.ico` - 应用图标（256x256 像素）

### macOS:
- `icon.icns` - 应用图标（需要包含多种尺寸）

### Linux:
- `icon.png` - 应用图标（512x512 像素）

## 如何获取图标

### 方法一：在线生成
1. 访问 https://www.icoconverter.com/
2. 上传您喜欢的图片（建议使用乌萨奇头像）
3. 选择输出格式（ICO、ICNS、PNG）
4. 下载并放到项目根目录

### 方法二：使用现有图标
1. 从网上下载喜欢的图标
2. 重命名为对应的文件名
3. 放到项目根目录

### 方法三：跳过图标
如果暂时没有图标，应用仍然可以正常运行，只是：
- Windows: 使用默认图标
- macOS: 使用默认图标
- Linux: 使用默认图标

## 推荐的图标

建议使用 Chiikawa 中乌萨奇的可爱头像作为应用图标，尺寸：
- 最小：256x256 像素
- 推荐：512x512 像素
- 最佳：1024x1024 像素

## 文件放置位置

将图标文件放在项目根目录：
```
test/
├── icon.ico      (Windows)
├── icon.icns     (macOS)
├── icon.png      (Linux)
── main.js
├── package.json
── ...
```

##  注意事项

- 图标文件不是必需的，应用没有图标也能正常运行
- 打包时如果找不到图标文件，会使用默认图标
- 可以在打包后再添加图标文件，重新打包即可
